# Medical React App

A minimal React + Vite app for viewing simple patient records and interacting with them (favorite, add notes).

Quick start

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Files
- `src/components/PatientList.jsx` - list and selection of patients
- `src/components/PatientDetails.jsx` - details and add note
- `src/data/patients.json` - sample data

Notes
- This is a local demo app without authentication or persistence.
- To add persistence, hook up a backend or use localStorage.
